Instructions

  1. Select database
  2. Select the aligner
  3. Select the input and output network. Type the name of the specie or select Personalized Network if you want to use your own data. To introduce your own network upload a text file with two columns of proteins such that each row will be an edge. Proteins' ID must be the same as the selected database.
  4. If you select a network from the String database, then you must select an interaction type and its threshold. For example, you can choose a neighborhood score with a threshold value of 600. Thus, those edges whose score are greater than 600 will be taken. To reset the threshold chooses the RESET ALL SCORES button.
  5. Write your email address to get the report file.
  6. Click send button
  7. You will receive a report with the results when it is done.
